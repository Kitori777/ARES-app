OBIEKTY / PODGLĄDY
Tutorial: Jak programować dodatki i skrypty w ARES
Praktyczny przewodnik od zera: API, komendy, gotowe przykłady i dobre praktyki.

1. Gdzie pisać kod

Wejdź w Dodatki → Skrypty tego arkusza, kliknij Nowy, nadaj nazwę i pisz kod JavaScript w polu „Kod skryptu”. Skrypt zapisuje się lokalnie do bieżącego arkusza.

2. Podstawowe komendy API

  • api.getCell("A1") – pobiera wartość komórki.
  • api.setCell("A1", "tekst") – wpisuje wartość do komórki.
  • api.getRange("A1:C3") – pobiera zakres danych.
  • api.setRange("A1:B2", [[1,2],[3,4]]) – wpisuje dane do zakresu.
  • api.getSelection() – zwraca bieżące zaznaczenie użytkownika.
  • api.notify("Gotowe") – pokazuje komunikat po zakończeniu.

3. Pierwszy skrypt (start)

const sel = api.getSelection() || "A1";
api.setCell(sel.split(":")[0], "✅ Działa");
api.notify("Skrypt uruchomiony poprawnie.");

4. Przykład: podsumowanie kolumny

const values = api.getRange("B2:B20");
let sum = 0;
for (const row of values) {
  const n = Number(row?.[0] || 0);
  if (!Number.isNaN(n)) sum += n;
}
api.setCell("B21", sum);
api.notify("Policzono sumę kolumny B.");

5. Dobre praktyki

  • Najpierw testuj na małym zakresie (np. 10–20 wierszy).
  • Dodawaj api.notify() po krytycznych krokach.
  • Nie nadpisuj całego arkusza, jeśli wystarczy fragment zakresu.
  • Po większych zmianach zapisz arkusz i sprawdź Historię zmian.

6. Workflow użytkownika

Dane (czyszczenie/sortowanie) → Dodatki (skrypt) → Raporty (analiza i PDF) → Historia (podgląd różnic i przywracanie wersji).
